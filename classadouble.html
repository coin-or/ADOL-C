Detailed Description

The adouble class is leveraged to compute tape-based derivatives. It is represented by a location on the tape and the value that is stored on the tape at the location.

Its interface acts in principle as double. However, internally, whenever it participates in an arithmetic operation, the adouble registers locations and the type of the operation on the tape.
