◆ zos_forward

ADOLC_ext_fct_v2* ext_diff_fct_v2::zos_forward

below are function pointers used for call back from the corresponding ADOL-C trace interpreters; these function pointers are initialized to 0 by reg_ext_fct; the user needs to set eplicitly the function pointers for the trace interpreters called in the application driver this points to a method implementing a forward execution of the externally differentiated function y=f(x); the pointer would typically be set to the same function pointer supplied in the call to reg_ext_fct, i.e. zos_forward would be equal to function (above) but there are cases when it makes sense for this to be different as illustrated in examples/additional_examples/ext_diff_func/ext_diff_func.cpp
